Cours Régulier

レギュラークラス

毎週同じ仲間と積み上げる、少人数の本格フランス語クラスです。

Regular Class

レギュラークラスとは

毎週固定の曜日・時間に通う少人数クラスです。最大4名の少人数制で、講師1名が一人ひとりのペースと目標に合わせて指導します。文法・読解・語彙・会話をバランスよく積み上げ、フランス語を着実に習慣にしたい方に向いています。

最大4名の少人数制完全オリジナルプリント教材入門〜中上級 レベル別6クラス欠席時は翌月月謝調整

Déroulement

レッスンの流れ

  1. 前回の復習・フォローアップ（5〜10分）
  2. 新テーマの導入（文法・語彙の解説）
  3. オリジナルプリントを使った読解・理解
  4. 会話練習・ロールプレイ
  5. 質疑応答・次回テーマ予告

Schedule

開催クラス・日程

全6クラス、レベル別に開講しています。入門から中上級まで対応。

月曜 10:15〜11:45（中級）月曜 13:30〜15:00（中上級・2026年4月〜）火曜 10:30〜11:50（入門）水曜 13:00〜14:20（基礎・2023年10月〜）土曜 11:00〜12:20（旅行フランス語・2017年〜）日曜 10:30〜12:00（社会人向け）

Tarifs

欠席時の調整翌月月謝を2,000円減額
体験レッスン無料（レベル確認後ご案内）
入会金なし
テキスト代なし（オリジナルプリント込み）

How to join

入会の流れ

①お問い合わせ（フォーム・メール・LINE）→ ②レベル確認（任意・15分）→ ③無料体験クラス受講 → ④翌週から本入会。入会義務はなく、体験後にゆっくりご検討いただけます。
