les vieux（年寄り）は失礼な言い方？年齢に関するフレーズ

vieux（ヴュウ）

というのはフランス語で「古い」という意味ですが、

年寄り

les vieux

若者

les jeunes

という言い方しか知らないと、気づかないうちに年配の方に失礼な言い方になってしまう時があるかもしれませんので、以下のような言い方も覚えておくと便利です。

年配の世代

les anciennes générations

若者世代

les jeunes générations

若く見える、をフランス語で

私は若く見える

Je fais jeune.

ジュ　フェ　ジョヌ

年齢の話題は日常会話で、ちょくちょく出てきますよね。

例えば、日本人、フランス人のみかけのちがいなど・・・

「日本人は若くみえる」というのはフランス人からよく聞きますし、そしてフランスに行けば、確かに日本人は若々しいなあと（時には子どもっぽいなあと）思う事がよくあります。

ヨーロッパの人々は老けてみえる

les européens font vieux.

レ　ズーろペアン　フォン　ヴィユー

老ける、老化する（動詞）

vieillir ヴィエイエ

気の毒、をフランス語で

気の毒ね、かわいそう、と同情する言い方です。

私はあなたを気の毒に思う。

Je vous plaint

ジュ　ヴ　プレン

それは残念ですね。

C’est dommage pour vous

セ　ド　マージュ　ポウ　ヴ

そう、残念だよ。（私は自分を気の毒に思う）

Je me plains.

ジュ　ム　プレン

羨ましい

Je  vous envie

ジュ　ヴ　ゾンヴィ

”通信講座”をフランス語で？

通信講座

cours par correspondance

クー　パー　コレスポンドンス

ゼロからスタートする

Commencer de zéro

会話サロンでは、フランス語の学習方法の話題もよく出ます。フランス語はいつから始めたんですか？どんなふうに勉強しているのですか？

そんな質問が参加者の方同士でよく飛び交いますし、講師からも質問したりします。

左利きには天才が多い、という話題

右利き

droitier ドロワティエ

左利き

gaucher ゴウシェ

両利き

ambidextre アンビデクストろ

フランスでも、左利きの子どもは強制的に右利きに修正されることがあるそうです。（現在はあまりないかもしれませんが）

元々は、何利きだったのですか？の「元々」は

à l’origine

と言います。

左利きの人は脳の使い方が右利きとは異なり、普通の人が思いつかないような、驚くべき才能を持っている、と言われています。

〜と言われている、と言いたい時は

on dit que オン　ディ　キュ

を最初に付けます。

違う能力

capacité différent

ちなみに、ダヴィンチ、アインシュタイン、ミケランジェロなどはみんな左利きだそうです。

左利きの人は、感情や創造性を司る右脳が優位である、と言われます。

右脳

la partie droite du cerveau

ラパるティ　ドろワット　ドゥ　サーヴォウ

または

l’hémisphère droit

l’hémisphère

レミスフェア

という単語は、北半球、南半球、という時に使いますよね。
